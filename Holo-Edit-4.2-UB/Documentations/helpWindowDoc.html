Holo-Edit Quick Help

Room Editor

Drawing Zone :

This is where you will create and manipulate all points and speakers on the X, Y and Z axis. 
Command Where Action
Command+Click (Mac)
Ctrl+Click (Win) 		on a point change point type (editable / non-editable)
on a line add a point on this line
else add a point at the end of the visible trajectory
add a speaker (if 'speakers' is checked)
Alt+Click on a point remove this point
on a speaker remove this speaker
2xClick select all visible points
Drag on a point / multiple points move
  • if the point is editable (large point), all non editable points between it and the previous/next one will move proportionally ; if not, only this current point will move.
  • if there's a selection, all points in the selection will move similarly
  • if Shift is down while dragging, the point/selection will only move along the Y axis.
  • if Command (Mac) or Ctrl (Win) is down while dragging, the point/selection will only move along the X axis.
on a speaker move
  • if Shift is down while dragging, the speaker will only move along the Y axis.
  • if Command (Mac) or Ctrl (Win) is down while dragging, the speaker will only move along the X axis.
else make a selection
  • by default the selection will contain all points between the first and the last editable points in terms of time.
  • if Shift is down while dragging, the selection will contain only points contained in the selection square.
Command+Shift+Drag (Mac)
Ctrl+Shift+Drag (Win) 		on a point /multiple points move point/selection up and down (Z)
on a speaker move this speaker up and down (Z)
Ctrl+Click or Right Click (Mac)
Right Click (Win)		 on a point manually edit this point
on a speaker manually edit this speaker
else contextual menu
Alt + Drag
drag the view
Mouse Wheel zoom In/Out around the cursor
If Shift is down, the zoom will be more accurate.
Z + Up/Down zoom In/Out around the center
If Shift is down, the zoom will be more accurate.
D + Up/Down/Left/Right displace the view
If Shift is down, the displacement will be more accurate.
  • Short view : while 'Short View' is checked, you'll only see tracks around active trajectory's end time +/- delta

Room Time Scale :

This scale shows you the current active track, it's content and the current time selection.
(Note that all time scales in Holo-Edit will respond the way this one does.)

Command Where Action
Click + Drag
 in the time selection Shift the time selection
  • if Shift is down, the time shift will be more accurate.
  • if Alt is down, create a new time selection.
on the begin/end of the time selection Shift the begin/end of the time selection.
  • if Shift is down, the time shift will be more accurate.
else Create a new temporal selection
  • if Shift is down, the time selection will be more accurate.
Left/Right Arrows Shifts the time selection from 'delta'

Transport :

Total (h:mn:s:cs)
indicate the total duration of the tracks
Begin (h:mn:s:cs)
indicate the time of beginning
End (h:mn:s:cs)
indicate the time of ending
Cursor (h:mn:s:cs)
indicate the time of the cursor in the score
Chrono (h:mn:s:cs)
indicate the current playing time
Delta (s:cs)
indicates the time between two new editable points (also used by the short view mode).
Points
indicates the number of points between two new editable points.

Time Editor :

Several curves are available (X, Y, Z, Ray, Theta, Sound, sdif data) and can be added or removed.
They are all manipulated the same way except the sound and sdif curves that cannot be manipulated.
The Time Editor shows trajectories in the time domain and allows a precise control over each parameters (X, Y, Z, ray, angle...).
It also shows sound waveforms and sdif representations for a precise synchronization between sound and space location.
Command Where Action
Command+Click(Mac)
Ctrl+Click (Win)		 on a point change point type (editable / non-editable)
else add a point
Alt+Click on a point / multiple points remove this point/selection
else set cursor time
Drag on a point / multiple points move selection (time)
Double-Click select all track
Ctrl+Click or Right Click (Mac)
Right Click (Win)		 on a point manually edit this point
Drag selection move selection, all points between the first and the last editable points of the selection will move.

Score :

Sounds, sdif datas, and trajectories can here be manipulated as objects in a musical score.
Sounds cannot be cut or stretched, only moved in time.
Command Where Action
Command+Click(Mac)
Ctrl+Click (Win)		 on a trajectory/sdif remove / cut at time
  • if shift is down, remove the trajectory/sdif from the score
  • if alt is down, cut both trajectory and sdif at that time
  • else cut trajectory/sdif at that time
Alt+Click nothing
 set cursor time
Drag on a trajectory/waveform
sdif/selection		 move selection
  • if alt is down, make a copy of the current waveform/trajectory/selection
  • if shift is down, the move will be more accurate
  • if ctrl is down when moving from a track to another, keep the same begin time for the selection
on the beginning/end of a trajectory/sdif move current sdif/trajectory's begin/end time by stretching it.
  • if alt is down, the sdif/trajectory is cut
  • if shift is down, the command is more accurate
else make a selection
  • if Command (Mac) or Ctrl (Win) is down, select only the trajectories
  • if Shift is down, select only the waveforms
  • else select both.
Double-Click on a trajectory/waveform/selection select this portion of time and this track and shows in the time editor
else select all content of this track
Ctrl+Click or Right Click (Mac)
Right Click (Win)		 on a trajectory/waveform different contextual menus
Drag waveform from soundpool add this waveform to destination track.
Drag sdif data from soundpool add this sdif data to destination track.
Drag soundfile from Finder/Explorer import this soundfile to soundpool and add it to the destination track.
Clicking on the track header responds to the same command as clicking on the track name in the tracks window (make it solo, see all, see all but one...)

Soundpool:

This is the waveform and sdif library.
Retrieves informations and graphic representations of imported sound and sdif files.
Holo-Edit will can only draw, move in time, and play soundfile. For further information see Documentation below.
Command Action
Drop a soundfile import this soundfile
Click on a soundfile
Shows the waveform of the sound and draws its attached sdif datas (if any).
Retrieves different informations.
Drop a SDIF file import this SDIF file
Click on a SDIF data
draws this SDIF data and retrieves different informations.
Drag a sdif data from the 'imported datas' zone
and drop it over a sound in the 'imported sound' zone		 attach this sdif data to the corresponding sound.
Ctrl+Click or Right Click (Mac)
Right Click (Win)		 contextual menu
Drag a waveform to the Score add this waveform to the destination track

Options :

  • Automatic track selection : if enabled, a track will be set active when the mouse roll over a point a this track.
  • View only edtable points : if enabled, only editable points will be selectable, removable...
  • View Loudspeakers : if disabled, loudspeakers won't be visible.
  • Coordinates (Cartesian/Polar) : employ this to select the types of coordinate which you prefer to work with. i.e. cartesian (x,y) or polar (R,T).
  • SDIF expert import : if enabled, allows you to define specific settings during SDIF file imports.
